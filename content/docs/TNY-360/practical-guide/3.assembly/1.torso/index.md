# The Torso

We'll start this assembly guide by building the torso, which is the core of the robot.

It will hold all the main PCBs, motors, and other power components (and a lot of wires!).

---

## Section overview

Here's a quick overview of the sections in this chapter:

- [**Adding the Components**](./components)<br>
    We'll add all the PCBs and otehr components on the torso, making sure they are all securely mounted.

- [**Adding the Motors**](./motors)<br>
    We'll add 4 of the 12 motors on the torso, which will be used for the Hip-Roll joints of the legs.

- [**Wiring Everything**](./wiring)<br>
    We'll wire up all the motors and PCBs together, making sure to connect everything to the right places and secure the cables properly.
