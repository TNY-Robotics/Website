# The Universal Mount

We're going to step up a bit the complexity of the part with the universal mount.

This part is the interface between the torso and the [Universal Mount](../../../development-modding/physical-extensions), which have to be securely fastened together to ensure a strong base for any extension.

---

## Sizes and placements

This part contains heat inserts for the [Universal Mount](../../../development-modding/physical-extensions) (on the top), but also for the [Universal Mount PCB](../../../anatomy/electronics/pcbs/mount) (on the bottom).

::split{side=right}
#left
For this part, you'll need these heat inserts:

| ID  | Size       | Description           | Quantity |
| --- | ---------- | --------------------- | -------- |
| 1   | M2.5 x 2mm | PCB Mounting Holes    | 2        |
| 2   | M2.5 x 4mm | Universal Mount Holes | 4        |

#right
<img src="/docs/images/insert/mount.png" class="w-2/3" />
::

## Placing the heat inserts

Place the heat inserts as described in the [Heat Inserts Guide](./index#heat-inserts-guide), making sure to place them in the correct holes as indicated in the image above.

## Final inspection

After placing the heat inserts, make sure to inspect the part to check that all the inserts are properly seated and flush with the surface of the part.

If you notice any issues, such as an insert that is not fully seated or is tilted, you can try to reheat it and adjust its position until it is properly placed.
