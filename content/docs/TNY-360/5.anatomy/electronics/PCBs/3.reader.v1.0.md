# Analog Reader

::split{side=right}
#left

The Analog Reader PCB is a communication interface between the analog sensors and the microcontroller.

This board works like a extension plug for the Main PCB, allowing to connect multiple analog sensors to the robot while keeping the wiring clean and organized.
