# The clip system

*Let's be honest ... Thats just satisfaying!*

The TNY-360 uses a clip system for all decorative parts of its body. This allows for easy assembly and disassembly of the parts, as well as decreasing the number of screws needed to build the robot.

Let's see how this clip system works, and how you can use it on the robot (or even on your own projects!).

---

## 1 - The clip design

All type of materials (including plastic) can deform slightly without breaking, and return to their original shape when the force is removed.
This property is called elasticity, and the clip system of the TNY-360 takes advantage of this property to hold the parts together.
