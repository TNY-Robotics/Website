# Main Board

::split{side=right}
#left

The Main Board is the central hub of the TNY-360's electronics.

It hosts the ESP32-S3 microcontroller, which is responsible for running the main firmware, handling communication with the sensors and other boards, and managing the overall operation of the robot.
