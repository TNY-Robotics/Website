# Products

TNY Robotics offers a range of products designed to meet the needs of robotics enthusiasts and professionals alike.

This section contains detailed information about our products, so you can make informed decisions about which ones are right for you.

---

## Product List

- **[TNY-360 | Maker Kit (Founder's Edition)](./tny-360-maker-kit-founders-edition.md)**<br>
  The TNY-360 Robot Dog Maker Kit (Founder's Edition) contains all the materials you need to build your own TNY-360 Robot Dog.

- **[TNY-360 | BareBones Kit (Founder's Edition)](./tny-360-barebones-kit-founders-edition.md)**<br>
  The TNY-360 Robot Dog BareBones Kit (Founder's Edition) contains all the components used in the TNY-360 robot. It does not include the 3D printed parts, which you can print yourself or purchase separately.
